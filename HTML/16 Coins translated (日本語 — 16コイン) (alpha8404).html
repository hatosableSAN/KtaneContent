4×4の正方形に16枚のコインが並べられる。「H」は頭(Heads)、「T」は尾(tails)である。モジュールを解除するには、手順に従ってターゲットコインを決定しクリックする。

  • 偶数の列の「H」の数を数える(下記に記された位置)。「H」の数が奇数の場合、ターゲットコインは偶数の列にある。それ以外の場合、ターゲットコインは奇数の列にある。
    x x
    x x
    x x
    x x
  • 右半分の「H」の数を数える(下記に記された位置)。「H」の数が奇数の場合、ターゲットコインは右半分にある。それ以外の場合、ターゲットコインは左半分にある。
    xx
    xx
    xx
    xx
  • 偶数の段の「H」の数を数える(下記に記された位置)。「H」の数が奇数の場合、ターゲットコインは偶数の段にある。それ以外の場合、ターゲットコインは奇数の段にある。
    o
    xxxx
    o
    xxxx
  • 下半分の「H」の数を数える(下記に記された位置)。「H」の数が奇数の場合、ターゲットコインは下半分にある。それ以外の場合、ターゲットコインは上半分にある。
    o
    o
    xxxx
